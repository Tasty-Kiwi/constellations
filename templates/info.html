Info

Markdown support

Constellations supports advanced markdown support, including support for tables, syntax highlighting and mermaid graphs.

Tables

Tables are created using GitHub's syntax:

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |

Graphs

Graphs are created using code block syntax:

```mermaid
graph TD;
  A-->B;
  A-->C;
  B-->D;
  C-->D;
```

More information on Mermaid syntax is in their website.
